# **RESTAURANT CHATBOT**

## Problem
Restaurants need a simple way for customers to place orders without complex apps or human intervention.

## Solution
A backend-driven chatbot that allows users to:
- View menu options
- Place food orders
- Confirm checkout
- Simulate payment
- Complete or cancel orders

## Features
- Chat-based interaction
- State-driven conversation flow
- Order creation and management
- Checkout and payment simulation
- No authentication required

## Chatbot States
- START
- MENU
- ORDERING
- CHECKOUT
- PAYMENT
- COMPLETED
- CANCELLED

## Tech Stack
- Node.js
- TypeScript
- Express.js
- REST API
